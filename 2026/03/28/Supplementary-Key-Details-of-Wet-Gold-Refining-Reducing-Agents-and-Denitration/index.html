湿法提纯黄金的关键细节补充：还原剂与去硝

发表于 2026 年 03 月 28 日 分类于 Mathematics and Natural Sciences 阅读次数：
本文字数： 1.4k 阅读时长 ≈ 3 分钟

前文已经把王水体系的主流程讲清楚了，但真正决定你能不能稳定做出高纯金的，其实不是那些「流程步骤」，而是两个细节：

  1. 你用什么还原剂（怎么把金「干净地」还原下来）
  2. 你有没有把硝处理干净（否则还原剂白给）

这篇试图讲清楚这俩问题。

一、还原剂：本质其实很简单

所有还原剂，本质都是「电子提供者」。金在溶液中是 Au³⁺（准确说是 AuCl₄⁻），要变成金属：

Au³⁺ + 3e⁻ → Au

所以问题就变成：谁来提供这 3 个电子？

二、三类典型还原剂对比（核心）

1）锌（Zn）：简单粗暴

反应本质

Zn → Zn²⁺ + 2e⁻

总体：
2AuCl₄⁻ + 3Zn → 2Au + 3Zn²⁺ + 8Cl⁻

特点

  • 优点：
    • 反应快
    • 几乎不会失败
  • 缺点：
    • 啥都还原（不挑食）
    • Cu、Fe、Pt 都可能一起还原沉淀下来
    • 引入 Zn 污染

总结来说，锌是「能用」，但提炼出来的金的纯度天花板较低。（3N 大约是极限）

2）亚硫酸体系（Na₂SO₃ / SO₂）

实际有效物

Na₂SO₃ 在酸中变成 SO₂：

Na₂SO₃ + 2HCl → 2NaCl + SO₂ + H₂O

然后：

2AuCl₄⁻ + 3SO₂ + 6H₂O → 2Au + 3SO₄²⁻ + 8Cl⁻ + 12H⁺

特点

  • 比锌「干净很多」
  • 但：
    • 对 pH 敏感（通常要求 pH 在 1 -- 3 这个区间）
    • SO₂ 会跑掉
    • 控制难度中等

总结来看，比锌好，但需要一定操作经验。

3）草酸（推荐）

反应

2AuCl₄⁻ + 3H₂C₂O₄ → 2Au + 6CO₂ + 8Cl⁻ + 6H⁺

特点

  • 副产物只有 CO₂（非常干净）
  • 几乎不带入杂质
  • 选择性好

不过，使用草酸作为还原剂有一个致命前提：必须先去硝；否则：

  • 草酸会被硝酸优先氧化
  • 从而，金根本下不来

总结来看： 草酸是「干净路线」的核心，但对前处理要求最高。

三、什么是「硝」？到底要去掉什么？

如此看来，去硝很重要。不过很多人理解错：

❌ 去硝 = 去掉 NO₃⁻
✔ 去硝 = 去掉「氧化能力」

王水体系里的真实情况

硝酸不会安静待着，而是在不断循环：

HNO₃ ⇌ NO₂ ⇌ NO ⇌ HNO₂ ⇌ NOCl

对于后续还原步骤来说，真正麻烦的是：NOx + HNO₂ 这个「活跃氧化循环」。它们的氧化性比 AuCl₄⁻ 要强得多，因此会优先与还原剂发生反应，从而一方面损耗更多还原剂，另一方面导致海绵金几乎不沉淀。

四、主流去硝方法（从可靠到不推荐）

1）蒸发 + 盐酸驱硝（最标准）

原理

HNO₃ + 3HCl → NOCl + Cl₂ + 2H₂O

即是说，把硝酸变成气体赶走。

操作逻辑（核心思路）

  • 加热蒸发
  • 补加 HCl
  • 再蒸发
  • 重复 1–2 次

优点

  • 最可靠
  • 不引入杂质
  • 同时调节酸碱度到合适的 pH 值，方便后续使用亚硫酸钠/草酸进行还原

缺点

  • 有 NO₂（必须通风）
  • 时间成本

这基本是工业和实验室的标准做法。

2）磺胺酸（强烈推荐的「收尾工具」）

是什么

NH₂SO₃H（磺胺酸 = 氨基磺酸）

核心反应

NH₂SO₃H + HNO₂ → N₂ + H₂SO₄ + H₂O

这本质是把 NOx / HNO₂ 直接变成无害的 N₂（气体）。

优点

  • 非常干净
  • 可控
  • 不引入有机杂质

如果追求完整地去硝，可以结合蒸发（去大头） + 磺胺酸（清尾巴）两种方法。从而真正「终止氧化循环」。

五、还原剂 vs 去硝：组合策略

低要求（快速回收）

  • 还原剂：Zn
  • 去硝：可忽略

👉 结果：低纯度

中等要求

  • 还原剂：SO₂ / Na₂SO₃
  • 去硝：简单蒸发

👉 结果：中等纯度

高纯度（推荐）

  • 还原剂：草酸
  • 去硝：
    • 蒸发 + HCl
    • 磺胺酸

👉 结果：高纯金（可达 3N–4N）

总结

整个湿法提金中，最容易被低估的不是「溶金」，你有没有把体系的「氧化能力」处理干净。还原剂决定「干不干净」，去硝决定「能不能成功」。
