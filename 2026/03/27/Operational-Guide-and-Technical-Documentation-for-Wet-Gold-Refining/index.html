湿法提纯黄金操作指南与技术说明（王水体系为主）

发表于 2026 年 03 月 27 日 更新于 2026 年 03 月 28 日 分类于 Mathematics and Natural Sciences 阅读次数：
本文字数： 1.9k 阅读时长 ≈ 3 分钟

最近金价飞涨，偶尔看到民间有提纯黄金的店铺发布的提纯过程的视频。沉睡多年的「化学」基因动了一下，于是总结了王水体系湿法提纯黄金（王水溶解 → 分离 → 选择性还原 → 酸洗 → 重复精炼 → 熔融成型）的简要操作指南和技术说明。此外，也简述硫脲（无氰）体系作为对照。

特别说明：纯属娱乐，操作风险极大，请勿在居民区操作。

一、整体流程（先有全局感）

可以把整个过程理解为 4 个循环步骤：

  1. 把金溶掉（进入溶液）
  2. 把金「干净地」还原出来
  3. 把杂质洗掉
  4. 必要时重复

最终再熔融得到金锭。

二、步骤详解（王水体系）

步骤 1：预处理（可选，但很重要）

目的

  • 去有机物（油、塑料、树脂）
  • 增大比表面积（让金更容易被溶解）

常见方法

  • 简单煅烧（去有机物）
  • 剪碎 / 粉碎（优先推荐）
  • 熔融后水淬（形成颗粒）

核心原则

让金尽量「薄、散、小」

步骤 2：配置王水

配方（标准）

  • 浓盐酸（HCl，约 36–38%）
  • 浓硝酸（HNO₃，约 65–68%）
  • 按体积比 3 : 1

例如：

  • 30 mL HCl + 10 mL HNO₃

王水本质

王水不是简单混酸，而是会生成活性物种：

HNO₃ + 3HCl → NOCl + Cl₂ + 2H₂O

👉 实际参与溶金的是：

  • Cl₂
  • NOCl
  • NOₓ

步骤 3：溶解金

核心反应逻辑

1）金被氧化：
Au → Au³⁺ + 3e⁻

2）被氯离子络合：
Au³⁺ + 4Cl⁻ → AuCl₄⁻

3）整体表达：
Au + HNO₃ + 4HCl → HAuCl₄ + NO + 2H₂O

现象

  • 溶液变黄 / 橙黄
  • 有气体（NO₂ / Cl₂）产生

关键注意

  • 王水必须现配现用
  • 必须通风（强烈）
  • 不要密闭容器（有爆炸风险）
  • 溶解不完全时，不要急着加很多硝酸（会带来后续问题）

步骤 4：过滤（关键分离）

滤液

  • 含金：主要是 AuCl₄⁻

滤渣可能包含

  • AgCl（最常见）
  • 未溶贵金属（Pt、Rh 等）
  • SnO₂ 胶体
  • 未反应金属核

特别提醒

AgCl 是这一阶段最重要的副产物（白色）

步骤 5：还原金（核心步骤）

这里是纯度的关键。

方案 A：锌（Zn）还原（简单但不推荐）

反应

2AuCl₄⁻ + 3Zn → 2Au↓ + 3Zn²⁺ + 8Cl⁻

副反应：
Zn + 2H⁺ → Zn²⁺ + H₂↑

特点

  • 优点：快、简单
  • 缺点：
    • 引入 Zn 杂质
    • 共还原 Cu、Fe、Pt 等
    • 得到「脏金粉」

方案 B：亚硫酸体系（更推荐）

实际有效还原剂是 SO₂：

Na₂SO₃ + 2HCl → 2NaCl + SO₂ + H₂O

还原反应：

2AuCl₄⁻ + 3SO₂ + 6H₂O → 2Au↓ + 3SO₄²⁻ + 8Cl⁻ + 12H⁺

条件

  • 必须在酸性环境（大致 pH 1–3）
  • 搅拌充分

特点

  • 杂质少
  • 颗粒细
  • 控制要求高

如何判断还原完成（重点）

唯一可靠方法：检测溶液中是否还有金

常用方法：

  • 氯化亚锡（SnCl₂）检测

结果：

  • 紫色/黑色 → 还有金
  • 无变化 → 基本还原完成

步骤 6：过滤（得到海绵金）

得到：

  • 棕色/黑色金粉（海绵金）

必须：

  • 充分洗涤（去 Cl⁻ / NO₃⁻）

步骤 7：酸解（去杂质）

目标：

把「混进金粉里的贱金属」溶掉

方案对比

1）稀硝酸（推荐）

反应示例：

Cu + 4HNO₃ → Cu²⁺ + 2NO₂↑ + 2H₂O + 2NO₃⁻
Zn + 4HNO₃ → Zn²⁺ + 2NO₂↑ + 2H₂O + 2NO₃⁻

特点：

  • 可溶 Cu / Zn / Fe
  • 提纯能力最强

2）稀盐酸

Zn + 2HCl → ZnCl₂ + H₂↑

特点：

  • 只能溶活泼金属（Zn / Fe）
  • 不溶 Cu

3）稀硫酸

Zn + H₂SO₄ → Zn²⁺ + SO₄²⁻ + H₂↑

特点：

  • 类似 HCl
  • 可能生成 PbSO₄ 等沉淀

推荐策略

  • Zn 还原 → 先 HCl，再 HNO₃
  • 高纯需求 → 直接 HNO₃

步骤 8：重复精炼（关键）

流程：

  • 再溶 → 再还原 → 再酸洗

每一轮：

  • 纯度显著提升

步骤 9：熔融成型

常加入：

  • 硼砂（Borax）

作用：

  • 吸附氧化物
  • 改善流动性

三、关于银和铂的处理

银（Ag）

主要存在：

  • AgCl（滤渣）

处理思路：

  • 转化 → 还原 → 得银粉/银块

铂（Pt）

注意：

  • 多数情况下在滤液中
  • 需单独沉淀（如铵盐法）

四、硫脲体系（对照说明）

原理

Au → Au⁺
Au⁺ + 2CS(NH₂)₂ → [Au(thiourea)₂]⁺

特点

项目 王水 硫脲
氧化方式 强氧化 温和
气体 NO₂ / Cl₂ 少
速度 快 慢
控制难度 中 高

本质区别

王水：强氧化 + Cl⁻络合
硫脲：络合主导 + 温和氧化

五、几个关键经验（非常重要）

1. 金的损失主要发生在：

  • 滤渣没洗干净
  • AgCl 包裹金
  • 沉淀太细被带走

2. 纯度的决定因素

  • 还原剂选择（决定「干净程度」）
  • 酸洗是否充分
  • 是否重复精炼

3. 王水的本质一句话

硝酸负责「氧化金」，盐酸负责「把金留在溶液里」

4. 最推荐路线（总结）

  • 王水溶解
  • 亚硫酸体系还原
  • 硝酸酸洗
  • 重复 1–2 次
  • 熔融

六、结语

这整套流程，本质上就是一句话：

把金「溶进去 → 干净地取出来 → 洗干净 → 再来一轮」

真正决定水平的，不是「有没有做这些步骤」，而是：

  • 每一步有没有做到干净、可控、可判断
